🔑 Obținere Supabase ID Token

Instrucțiuni:
  1. Deschide aplicația SuperParty într-un alt tab
  2. Autentifică-te în aplicație
  3. Revino aici și apasă butonul de mai jos
⚠️ Dacă butonul nu funcționează:

1. Deschide aplicația SuperParty într-un alt tab

2. Apasă F12 (sau Cmd+Option+I pe Mac) pentru Developer Tools

3. Mergi la tab-ul "Console"

4. Copiază și lipește această comandă:

supabase.auth().currentUser.getIdToken().then(token => { console.log('TOKEN:', token); prompt('Copiază token-ul:', token); });
